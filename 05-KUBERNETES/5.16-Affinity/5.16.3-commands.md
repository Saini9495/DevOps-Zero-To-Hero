# Affinity Commands

## Check Node Labels

```bash
kubectl get nodes --show-labels
```

## Add Label to Node

```bash
kubectl label node worker1 disktype=ssd
```

## Remove Label

```bash
kubectl label node worker1 disktype-
```

## Describe Node

```bash
kubectl describe node worker1
```

## Check Pod Placement

```bash
kubectl get pods -o wide
```

## Describe Pod

```bash
kubectl describe pod nginx
```

## Apply Affinity YAML

```bash
kubectl apply -f affinity.yml
```

## Delete Resource

```bash
kubectl delete -f affinity.yml
```

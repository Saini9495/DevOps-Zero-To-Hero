# Prometheus

## What is Prometheus?

Prometheus is an open-source monitoring and alerting tool used to collect and store metrics from applications, servers, containers, and Kubernetes clusters.

---

## Why Use Prometheus?

- Real-time monitoring
- Time-series database
- Powerful query language (PromQL)
- Kubernetes integration
- Alert generation

---

## Features

- Metrics Collection
- Time Series Database
- Alerting
- Service Discovery
- Visualization Support

---

## Architecture

Application
    |
Exporter
    |
Prometheus Server
    |
Time Series Database
    |
Grafana Dashboard

---

## Components

### Prometheus Server

Collects and stores metrics.

### Exporters

Collect metrics from services.

Examples:

- Node Exporter
- MySQL Exporter
- Redis Exporter
- Blackbox Exporter

### AlertManager

Handles alerts.

### Grafana

Visualizes metrics.

---

## Metric Types

### Counter

Only increases.

Example:

Total HTTP Requests

### Gauge

Can increase and decrease.

Example:

CPU Usage

### Histogram

Measures distribution.

Example:

Request Duration

### Summary

Measures latency and quantiles.

---

## PromQL Examples

CPU Usage

```promql
node_cpu_seconds_total
```

Memory Usage

```promql
node_memory_MemAvailable_bytes
```

Request Rate

```promql
rate(http_requests_total[5m])
```

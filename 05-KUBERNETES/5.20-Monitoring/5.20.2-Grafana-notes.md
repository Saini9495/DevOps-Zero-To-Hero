# Grafana

## What is Grafana?

Grafana is an open-source visualization tool used to create dashboards from monitoring data.

---

## Why Use Grafana?

- Dashboard Creation
- Real-time Monitoring
- Alerting
- Multiple Data Sources

---

## Architecture

Prometheus
    |
Grafana Data Source
    |
Dashboard
    |
Visualization

---

## Features

- Dashboards
- Alerts
- Panels
- Graphs
- Heatmaps

---

## Supported Data Sources

- Prometheus
- Elasticsearch
- MySQL
- PostgreSQL
- Loki

---

## Dashboard Components

### Panel

Single graph/chart.

### Row

Collection of panels.

### Dashboard

Collection of rows and panels.

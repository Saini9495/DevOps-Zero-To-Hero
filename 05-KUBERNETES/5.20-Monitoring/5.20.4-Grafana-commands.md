# Grafana Commands

## Deploy Grafana

```bash
kubectl apply -f grafana-deployment.yml
```

## Verify Pods

```bash
kubectl get pods -n monitoring
```

## Verify Services

```bash
kubectl get svc -n monitoring
```

## Port Forward

```bash
kubectl port-forward svc/grafana 3000:3000 -n monitoring
```

Access:

http://localhost:3000

Default Login:

Username: admin

Password: admin

## Logs

```bash
kubectl logs deployment/grafana -n monitoring
```

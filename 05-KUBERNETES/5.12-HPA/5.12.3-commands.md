# HPA Commands

## Check Metrics Server

```bash
kubectl get pods -n kube-system
```

## View CPU Usage

```bash
kubectl top pods
```

## Create HPA

```bash
kubectl apply -f hpa.yml
```

## Get HPA

```bash
kubectl get hpa
```

## Describe HPA

```bash
kubectl describe hpa nginx-hpa
```

## Watch HPA

```bash
kubectl get hpa -w
```

## Check Deployment

```bash
kubectl get deployment
```

## Check Pods

```bash
kubectl get pods -w
```

## Delete HPA

```bash
kubectl delete hpa nginx-hpa
```

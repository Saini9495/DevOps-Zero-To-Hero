# Kubernetes Service Commands

## Create Service

```bash
kubectl apply -f clusterip.yml
kubectl apply -f nodeport.yml
kubectl apply -f loadbalancer.yml
```

---

## List Services

```bash
kubectl get svc
```

or

```bash
kubectl get services
```

---

## Detailed Information

```bash
kubectl describe svc nginx-clusterip
```

---

## Delete Service

```bash
kubectl delete svc nginx-clusterip
```

---

## Check Endpoints

```bash
kubectl get endpoints
```

---

## Port Forward

```bash
kubectl port-forward service/nginx-clusterip 8080:80
```

---

## Service DNS

```bash
kubectl exec -it pod-name -- nslookup nginx-clusterip
```

---

## Watch Services

```bash
kubectl get svc -w
```

---

## Get Service YAML

```bash
kubectl get svc nginx-clusterip -o yaml
```

---

## Edit Service

```bash
kubectl edit svc nginx-clusterip
```

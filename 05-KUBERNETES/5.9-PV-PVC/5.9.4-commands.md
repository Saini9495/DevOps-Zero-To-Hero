# PV PVC Commands

## Create PV

```bash
kubectl apply -f pv.yml
```

## Create PVC

```bash
kubectl apply -f pvc.yml
```

## Verify PV

```bash
kubectl get pv
```

## Verify PVC

```bash
kubectl get pvc
```

## Describe PV

```bash
kubectl describe pv nginx-pv
```

## Describe PVC

```bash
kubectl describe pvc nginx-pvc
```

## Check Binding

```bash
kubectl get pv,pvc
```

STATUS = Bound
```

## Create Pod

```bash
kubectl apply -f pod.yml
```

## Verify Pod

```bash
kubectl get pods
```

## Delete PVC

```bash
kubectl delete pvc nginx-pvc
```

## Delete PV

```bash
kubectl delete pv nginx-pv
```

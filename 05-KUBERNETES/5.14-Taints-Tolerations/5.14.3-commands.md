# Taints and Tolerations Commands

## View Nodes

```bash
kubectl get nodes
```

## Describe Node

```bash
kubectl describe node worker-node
```

## Add Taint

```bash
kubectl taint nodes worker-node env=production:NoSchedule
```

## Remove Taint

```bash
kubectl taint nodes worker-node env=production:NoSchedule-
```

## Check Taints

```bash
kubectl describe node worker-node | grep Taint
```

## Create Pod

```bash
kubectl apply -f taint.yml
```

## View Pods

```bash
kubectl get pods -o wide
```

## Describe Pod

```bash
kubectl describe pod nginx-taint-demo
```

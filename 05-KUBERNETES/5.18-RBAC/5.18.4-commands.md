# RBAC Commands

## Create Namespace

```bash
kubectl create namespace dev
```

## Create Service Account

```bash
kubectl create serviceaccount app-user -n dev
```

## Create Role

```bash
kubectl apply -f role.yml
```

## Create RoleBinding

```bash
kubectl apply -f rolebinding.yml
```

## Verify Role

```bash
kubectl get role -n dev
```

## Verify RoleBinding

```bash
kubectl get rolebinding -n dev
```

## Describe Role

```bash
kubectl describe role pod-reader -n dev
```

## Check Permissions

```bash
kubectl auth can-i get pods --as=system:serviceaccount:dev:app-user -n dev
```

## Delete Resources

```bash
kubectl delete role pod-reader -n dev
kubectl delete rolebinding pod-reader-binding -n dev
```

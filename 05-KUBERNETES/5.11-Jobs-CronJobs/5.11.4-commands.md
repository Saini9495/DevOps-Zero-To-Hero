# Kubernetes Job Commands

## Create Job

```bash
kubectl apply -f job.yml
```

## List Jobs

```bash
kubectl get jobs
```

## Describe Job

```bash
kubectl describe job demo-job
```

## View Logs

```bash
kubectl logs <pod-name>
```

## Delete Job

```bash
kubectl delete job demo-job
```

# Kubernetes CronJob Commands

## Create CronJob

```bash
kubectl apply -f cronjob.yml
```

## List CronJobs

```bash
kubectl get cronjobs
```

## Describe CronJob

```bash
kubectl describe cronjob demo-cronjob
```

## View Generated Jobs

```bash

```

## View Pods

```bash
kubectl get pods
```

## Delete CronJob

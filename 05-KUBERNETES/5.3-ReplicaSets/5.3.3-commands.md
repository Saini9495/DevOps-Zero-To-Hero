# ReplicaSet Commands

## Create ReplicaSet

```bash
kubectl apply -f replicaset.yml
```

---

## Get ReplicaSets

```bash
kubectl get rs
```

OR

```bash
kubectl get replicasets
```

---

## Describe ReplicaSet

```bash
kubectl describe rs nginx-rs
```

---

## Get Pods

```bash
kubectl get pods
```

---

## Get Pods with Labels

```bash
kubectl get pods --show-labels
```

---

## Delete a Pod

```bash
kubectl delete pod <pod-name>
```

ReplicaSet automatically creates a new Pod.

---

## Scale ReplicaSet

```bash
kubectl scale rs nginx-rs --replicas=5
```

---

## Edit ReplicaSet

```bash
kubectl edit rs nginx-rs
```

---

## View ReplicaSet YAML

```bash
kubectl get rs nginx-rs -o yaml
```

---

## Delete ReplicaSet

```bash
kubectl delete rs nginx-rs
```

---

## Delete ReplicaSet and Pods

```bash
kubectl delete rs nginx-rs --cascade=foreground
```

---

## Watch ReplicaSet Changes

```bash
kubectl get rs -w
```

---

## Watch Pods

```bash
kubectl get pods -w
```

---

## Check Events

```bash
kubectl get events
```

---

## Useful Shortcuts

```bash
kubectl get rs
kubectl get pods
kubectl describe rs nginx-rs
kubectl scale rs nginx-rs --replicas=5
kubectl delete rs nginx-rs
```
